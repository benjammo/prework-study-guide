Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The metadata is invisible to the viewer, and contains information for the web browser, width of device, and for search engines.
  • Charset UTF-8 is designed as a superior alternative to UTF-1, including partial ASCII compatibility. Yay!

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A style sheet allows for DRY logic, basically meaning single lines of styling can apply to an entire page instead of individually in each element.
